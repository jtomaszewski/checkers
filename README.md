Checkers in C++11

My assumptions while creating the game:
* maximum clarity of the code
* easy to maintain
* DRY

How to build:
* linux required
* g++ with c++11 version and allegro5 libraries
* compiling: `cd Release/; make all;`
* running: run `./Release/checkers` while being in ./ directory
